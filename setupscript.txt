# First Run Setup

1. Install Node.js (v18 or later) and npm.
2. From the project root run:
   ```bash
   npm install
   ```
3. Optionally create a `.env` file and provide `VITE_GOOGLE_MAPS_KEY` if you use the map viewer in `public/game`.
4. To verify the code style, execute:
   ```bash
   npm run lint
   ```
5. Start the development server with:
   ```bash
   npm run dev
   ```
